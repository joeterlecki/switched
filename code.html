The Code

The function getString receives the string from the browser based on a button click event.

The first step is to ensure that the alert cards are invisible and get the string from the normalString HTML ID.

Once the string is received, the functions reverseString and displayString execute to reverse the input and display it to the DOM respectively.

The function reverseString takes the input and loops over the string in reverse order. This is completed by applying a -1 of the starting index. Then for the length of the string, which is tracked by using the length method and iteratively decrementing the index .

This is repeated until the whole string has been looped over. As each character is looped over, it is appended to switched string array. After the last character has been appended, the loop exits and returns.

The function displayString recieved the output of reverseString and displayed the results by getting the message id from the HTML and adding the value using a template literal to the DOM.

Finally, immediately after the message is added to the page, the class 'invisible' is removed from the alert using the classList.Remove method.


              function getString() {
                document.getElementById("alert").classList.add("invisible");
                let normalString = document.getElementById("normalString").value;
                let switchedString = reverseString(normalString);
              
                dispalyString(switchedString);
              
              }
              
              function reverseString(normalString) {
                let switchedString = [];
              
                for (let character = normalString.length - 1; character >= 0; character--) {
                  switchedString += normalString[character];
                }
              
                return switchedString;
              
              }
              
              function dispalyString(switchedString) {
              
                document.getElementById("message").innerHTML = `${switchedString}`;
              
                document.getElementById("alert").classList.remove("invisible");
              }
